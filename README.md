# Paroxysm Launcher
A high-performance, bloat-free Electron-based launcher designed for power users. Focus on speed, security, and a pure "tech-aesthetic" experience. No ads, no clutter, just the summit of your game. Features Modrinth integration &amp; advanced instance management. ⚡
