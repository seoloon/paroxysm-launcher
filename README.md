# Paroxysm Launcher
A high-performance, bloat-free Electron-based launcher designed for power users. Focus on speed, security, and pure experience. No ads, no clutter, just the summit of your game. Features Modrinth integration &amp; advanced instance management. ⚡

Actively working on it, release planned for May 2026.

Official website : https://paroxysm.seoloon.work/
